======================================================
     네이버 플레이스 순위 검색 웹앱 실행 안내서
======================================================

이 프로그램은 네이버 플레이스 키워드 검색 시 특정 매장이 몇 위에 노출되는지 확인하는 웹 프로그램입니다. 처음 실행하시는 분은 아래 순서대로 세팅해 주세요.

[ 1. 준비 사항 (파이썬 설치) ]
1. 컴퓨터에 Python이 설치되어 있어야 합니다.
2. 설치되어 있지 않다면 파이썬 공식 홈페이지(https://www.python.org/downloads/)에서 최신 버전을 다운로드하여 설치해 주세요.
   ※ 중요: 설치 시 첫 화면 하단에 있는 [ Add Python to PATH ] 체크박스를 반드시 체크하고 설치하셔야 합니다.

[ 2. 파일 확인 ]
압축을 푸신 폴더 안에 아래 파일들이 모두 있는지 확인해 주세요.
- app.py (실행 파일)
- naver_searchad_keyword_volume.py
- requirements.txt
- templates/ 폴더 (안에 index.html, details.html)

[ 3. 필수 라이브러리 설치 ]
프로그램 실행에 필요한 부품(라이브러리)들을 설치하는 과정입니다.
1. 키보드의 [ Windows 키 + R ]을 누르고 'cmd'를 입력해 명령 프롬프트(검은 창)를 엽니다.
2. 명령 프롬프트에서 압축을 푼 폴더 경로로 이동합니다.
   (예: cd C:\Users\user\Downloads\웹앱폴더)
3. 아래 명령어를 복사해서 붙여넣고 엔터를 칩니다.
   pip install -r requirements.txt

[ 4. 내부 브라우저(Playwright) 설치 ]
이 프로그램은 순위를 검색하기 위해 내부적으로 크롬 브라우저를 사용합니다.
1. 방금 열어둔 명령 프롬프트 창에 아래 명령어를 입력하고 엔터를 칩니다.
   playwright install chromium
2. 설치가 완료될 때까지 잠시 기다려주세요.

[ 5. 프로그램 실행 방법 ]
모든 세팅이 끝났습니다! 이제 프로그램을 실행합니다.
1. 명령 프롬프트 창에 아래 명령어를 입력하고 엔터를 칩니다.
   python app.py
2. 검은 창에 "Running on http://0.0.0.0:5000" 또는 "http://127.0.0.1:5000 으로 접속하세요" 라는 문구가 뜨면 성공입니다. (이 검은 창은 프로그램을 사용하는 동안 닫으시면 안 됩니다.)
3. 평소에 쓰시는 인터넷 브라우저(크롬, 엣지 등)를 열고 주소창에 아래 주소를 입력하세요.
   http://127.0.0.1:5000
4. 웹 화면이 나타나면 프로그램을 사용하시면 됩니다!

[ ※ 참고 사항 및 오류 해결 ]
- 프로그램을 종료하시려면 띄워두신 검은 창(명령 프롬프트)을 그냥 닫으시거나, 창 안에서 [ Ctrl + C ]를 누르시면 됩니다.
- "pandas가 설치되지 않았습니다" 라는 경고가 뜨면 `pip install pandas openpyxl` 명령어를 입력해 추가로 설치해 주세요 (엑셀 파일 업로드 시 필요).
- 기존에 크롬 창이 너무 많이 열려있거나 시스템 메모리가 부족하면 검색 중 에러가 발생할 수 있습니다. 가급적 불필요한 프로그램은 닫고 실행해 주세요.
